+ [VS Code]()

### Server

+ Neofetch
+ Docker

<br>

### Como rodar :

1. Clone o repositório e abra o diretório

~~~
$ git clone https://github.com/adelsonsljunior/pos-instalacao-ubuntu.git
$ cd pos-instalacao-ubuntu/
~~~

2. Escolha qual script de instalação deseja usar

+ Desktop
~~~
$ cd desktop/
$ chmod +x script.sh
$ ./script.sh
~~~

+ Server

~~~
$ cd server/
$ chmod +x script.sh
$ ./script.sh
~~~
